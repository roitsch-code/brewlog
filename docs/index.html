The idea

Not a product.
A personal tool.

Every other coffee app tries to make everyone happy. BrewLog tries to make one person happy — whoever built it. The equipment list, the grind settings, the taste baseline, the recommendation logic: all of it is configured around a single person's setup and palate. That is the point. And that is what became possible with tools like Claude Code.

Built in three evenings. A €20 Claude Code account. No team. No roadmap. No compromise.

End to end

Every interaction
creates data.

BrewLog is not a feature set. It is a loop. Every part of it — every tap, every rating, every coffee shop visit — is a data occasion that makes the next recommendation sharper.

The feedback loop
Scan a bag → Get a recipe → Brew it → Log taste + craft → Sharper next time
